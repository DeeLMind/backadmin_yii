backadmin_yii
=============

php后台，在工作中总有一些后台数据管理的工作，此处抽取出一个数据后台框架，方便其他项目迅速搭建后台； 功能包括用户，登录，权限，功能点（路由）的管理，后台基础功能健全。

外观样式支持横向菜单和纵向菜单；

简化后台可取消权限限制，无需用户登录；



后台基于php框架yii，以及模板metronic

后台示例：http://backadmin.hucong.net/site/index  用户名:admin 密码:admin
